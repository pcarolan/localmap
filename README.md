# LocalMap — Petoskey, MI

Interactive map showing local places, businesses, events, and news in Petoskey, Michigan.

## Development

Open `index.html` in a browser. No build step required.

For local development with data loading, serve with a local HTTP server:

```bash
python3 -m http.server 8080
```

## Hosting

Deployed to GitHub Pages from the `main` branch.
